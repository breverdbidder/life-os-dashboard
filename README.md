# Shapira Life OS Dashboard

ADHD-optimized productivity dashboard powered by Claude AI and Supabase.

## Features
- Real-time task tracking with 7-state machine
- ADHD intervention monitoring
- Dual timezone display (FL/IL)
- Domain-based organization (Business, Michael D1, Family, Personal)
- Completion streaks and gamification

## Quick Start

1. Clone this repo
2. Copy `.env.example` to `.env.local` and add your Supabase keys
3. Run `npm install`
4. Run `npm run dev`

## Environment Variables

```
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_anon_key
```
